Here's a basic guide to help you before starting the game:

  1. Open the game.
  2. Press the start button to play the game.
  3. Shoot the targets as fast as you can.
  4. Once the time limit is up, check your rate of targets per second.
  5. After finishing the game, press 'reset game' in case you wish to try again.

tips

And tRicks

In reality, there aren't really any tips for this game except practice. You will get the hang of it when you keep on playing it but we will just put tips on practicing.

  1. Keep one sensitivity. You can change the DPI and mouse sensitivity on your computer but once you feel comfortable with it, make sure to keep it that way so it will be easier to get used to.
  2. Start slow. The aim of the game is to click as much circles in the short time span but that doesn't mean you have to rush it. Make sure that you are clicking the targets accurately so you would improve a lot more.
  3. Concentrate. This is really self explanatory. You need to concentrate when playing this game so you can get the highest score you can possibly achieve.
  4. Rest when needed. You will not achieve anything if you don't rest when you feel tired, you will just get tired and play poorly so don't forget to rest when you really need it.
  5. Have Fun! There is no competition on this game so just make sure to have fun. We made this game because we played similar games related to these ones and some of us play these games to improve or lessen the stress.
